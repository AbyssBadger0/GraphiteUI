# V1｜成长闭环流 (Growth Loop Strategy) 图片分镜脚本

- 核心定位：利用极简点击交互降低门槛，通过视觉尺度的指数级扩张展示SLG的成长成就感。
- 视觉风格：等距视角 (Isometric)，高保真写实材质，强调建筑从木质到石质的质感演变，光影效果强烈。
- 前3秒钩子：模拟手指点击金矿，伴随即时的数值跳动反馈，营造解压感。
- 核心冲突：从微小的资源积累到宏大的帝国扩张与征服。
- 卖点：极简点击交互 / 建筑形态质变 / 视觉尺度跨越
